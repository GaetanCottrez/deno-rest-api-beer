## run the app
`deno run --allow-env --allow-net app.ts`

## routes

| Route                | Description              |
| -------------------- | ------------------------ |
| GET /beers           | return all the beers     |
| GET /beers/{name}    | return a beer            |
| POST /beers          | add a beer               |
| PUT /beers/{name}    | update a beer            |
| DELETE /beers/{name} | delete a beer            |
